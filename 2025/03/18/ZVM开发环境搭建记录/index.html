2.1Ubuntu虚拟机运行Zephyr

2.1.1一些概念

（1）west是 Zephyr 项目所采用的一个命令行工具，主要用于管理多个 Git 仓库组成的项目，也就是所谓的多仓库工作空间。west.yml文件则定义了这些仓库的组织方式、版本信息以及它们之间的依赖关系等内容。

（2）kitware archive 软件项目可能依赖于一些特定的库或工具，而这些资源可能不在系统默认的软件源中。Kitware 公司为了方便用户获取和使用其相关软件，会提供一个专门的归档文件或资源库，这里就是通过 “Kitware archive script”（Kitware 归档脚本）来将 Kitware 的 APT（Advanced Packaging Tool）软件源添加到系统的软件源列表中，以便能够从该源下载和安装满足项目依赖所需的软件包。

（3）python的venv包用于创建 Python 虚拟环境，安装项目单独需要的一些依赖项，同时与其他项目隔离。在激活的虚拟环境中，所有的 Python 包安装和使用都将局限于该虚拟环境，不会影响系统全局的 Python 环境。
https://docs.zephyrproject.org/latest/develop/getting_started/index.html#troubleshooting-installation

事实上，不是什么太重要的概念只是之前没见过所以记一下……
